# S15-S16ProjectBreak2
Tienda de ropa con catálogo de productos y dashboard para el administrador.

PASOS PARA INICIAR.

1. npm install
2. conectar base de datos con .env (MONGO_URI)
3. npm start.

ENDPOINTS:

productRoutes:
/product : rutas para ver el catalogo y filtrar por categorías pero sin poder modificar nada.
/dashboard : rutas para el administrador 

apiRoutes:
/API : rutas para API con datos json

authRoutes:
/login : para iniciar sesión
/registro : para registrar usuario 


RECURSOS:
-Express
-Mongoose
-Atlas
-Render
-dotenv
-express.urlencoded
-cloudinary
-method-override
